<p align="center">Node.js boilerplate with a DDD approach, built on top of Nest.</p>

## Stack
Node.js <br />
TypeScript <br />
Express  <br />
MongoDB

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
